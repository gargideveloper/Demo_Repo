Git Commands


Command Description
git status
git init
git add -a to add files into staging stage
git commit -m committed with message
git log to see committed log
git add(file name) to send file into staging stage
rm -rf .git deleting git repo
git diff it will compare staging and working directory
git commit -a -m "" to add all staged files into commit
git rm(file name) to remove files from git repo (it will also stage your removed file)
git mv(file name) to rename files
git rm --cached (file name ) to remove file from staging area
git log -p to view all commit log with insertion and deletion
git log -p -1 to view log of last commit added
git commit --amend to ammend or merge changes in last commit
